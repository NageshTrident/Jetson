    DISTRIBUTION KIT:   XCAP for the PIXCI(R) Imaging Boards
                        Version 3.8

    The XCAP distribution contains:


    1a) XCAPWXX.EXE         Startup executable, for Windows.

    1b) xcaplnx             Startup executable, for Linux.


    2a) EPIXXCW5.VXD        Windows 95/98/ME 32 bit driver.
        EPIXXCW5.INF        Windows 95/98/ME 32 bit driver installer.
        EPIXXCWT.SYS        Windows NT 32 bit driver.
        EPIXXCWT.INF        Windows NT 32 bit driver installer.
        EPIXXCW2.SYS        Windows 2000/XP/Vista/7/8/10 32-Bit driver.
        EPIXXCW2.INF        Windows 2000/XP/Vista/7/8/10 32-Bit driver installer.
        EPIXXCW2.CAT        Windows 2000/XP/Vista/7/8/10 32-Bit driver catalog.
        EPIXXCW6.SYS        Windows XP/Vista/7/8/10/11 64-Bit driver.
        EPIXXCW6.INF        Windows XP/Vista/7/8/10/11 64-Bit driver installer.
        EPIXXCW6.CAT        Windows XP/Vista/7/8/10/11 64-Bit driver catalog.

    2b) EPIXXC.SYS          Large image memory reservoir driver for Windows 95/98.

    2c) pixci_i386.ko       Loadable device driver for Linux w. 32 bit kernel
                            and Intel 386 and newer processors.

    2d) pixci_x86_64.ko     Loadable device driver for Linux w. 64 bit kernel
                            and Intel x86-64 processors.

    2e) pixci_armv7l.ko     Loadable device driver for Linux w. 32 bit kernel
                            and nVidia TK1 (ARM Cortex-A15).

    2f) pixci_aarch64.ko    Loadable device driver for Linux w. 64 bit kernel
                            and nVidia Nano/TX1/TX2/Xavier/XavierNX (ARM Cortex-A57).

    2g) pixcitty_i386.ko    Loadable device driver for /dev/tty serial access,
        pixcitty_x86_64.ko  for Linux w. 32 bit kernel and Intel 386 and newer processors,
        pixcitty_aarch64.ko or Linux w. 64 bit kernel and Intel x86-64 processors,
        pixcitty_armv7l.ko  or Linux w. 64 bit kernel and nVidia Nano/TX1/TX2/Xavier/XavierNX (ARM Cortex-A57),
                            or Linux w. 32 bit kernel and nVidia TK1 (ARM Cortex-A15).

    2h) pixciv4l_i386.ko    Loadable device driver for V4L2 access,
        pixciv4l_x86_64.ko  for Linux w. 32 bit kernel and Intel 386 and newer processors,
        pixciv4l_aarch64.ko or Linux w. 64 bit kernel and Intel x86-64 processors,
        pixciv4l_armv7l.ko  or Linux w. 64 bit kernel and nVidia Nano/TX1/TX2/Xavier/XavierNX (ARM Cortex-A57),
                            or Linux w. 32 bit kernel and nVidia TK1 (ARM Cortex-A15).

    2i) i386/src_*/pixcipub.h               Driver compilation kit, allowing re-compilation
        i386/src_*/pixcipub.c               in the field under the current kernel.
